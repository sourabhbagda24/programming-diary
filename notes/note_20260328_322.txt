4. 💡 DAILY CODE PROBLEM
   Problem title: "Container With Most Water"
   Difficulty: Medium
   Problem statement: Given n non-negative integers a1, a2, ..., an, where each represents a point at coordinate (i, ai). 
   n vertical lines are drawn such that the two endpoints of the line i is at (i, ai) and (i, 0). 
   Find two lines, which, together with the x-axis forms a container, such that the area of water it contains is maximized.
   Example input: [1,8,6,2,5,4,8,3,7]
   Example output: 49
   My approach and thought process: 
   To solve this problem, I used a two-pointer technique, where one pointer starts at the beginning of the array and the other at the end. 
   I calculated the area of water that can be trapped between the two lines at each position, and then moved the pointer that corresponds to the shorter line towards the other pointer. 
   This ensures that we are always considering the maximum possible area at each step. 
   Here is the full working solution in Python:
   ```python
def maxArea(height):
    max_area = 0
    left = 0
    right = len(height) - 1
    while left < right:
        width = right - left
        height_min = min(height[left], height[right])
        area = width * height_min
        max_area = max(max_area, area)
        if height[left] < height[right]:
            left += 1
        else:
            right -= 1
    return max_area
   ```
   Time complexity: O(n), where n is the number of lines
   Space complexity: O(1), as we only use a constant amount of space
   This problem was a great exercise in thinking creatively and using the right data structures and algorithms to solve a complex problem.
