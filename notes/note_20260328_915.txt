4. 💡 DAILY CODE PROBLEM 
**Problem Title:** Valid Palindrome
**Difficulty:** Easy
**Problem Statement:** Given a string, determine if it is a palindrome, considering only alphanumeric characters and ignoring case.
**Example Input and Output:** 
Input: "A man, a plan, a canal: Panama"
Output: true
Input: "Not a palindrome"
Output: false

My approach to this problem was to use a two-pointer technique, where I start from the beginning and end of the string and move towards the center. I ignore any non-alphanumeric characters and compare the characters at the current positions. If I find any pair of characters that don't match, I return false. If I successfully compare all pairs of characters without finding any mismatches, I return true.

Here is the full working solution in JavaScript:
```javascript
function isPalindrome(s) {
    s = s.toLowerCase();
    let left = 0;
    let right = s.length - 1;
    while (left < right) {
        if (!isAlphanumeric(s[left])) {
            left++;
        } else if (!isAlphanumeric(s[right])) {
            right--;
        } else {
            if (s[left] !== s[right]) {
                return false;
            }
            left++;
            right--;
        }
    }
    return true;
}

function isAlphanumeric(c) {
    return (c >= 'a' && c <= 'z') || (c >= '0' && c <= '9');
}
```
The time complexity of this solution is O(n), where n is the length of the string, and the space complexity is O(1), since we only use a constant amount of space.
