### 💡 DAILY CODE PROBLEM
**Problem Title:** Maximum Subarray
**Difficulty:** Easy
**Problem Statement:** Given an array of integers, find the contiguous subarray with the largest sum.
**Example Input and Output:**
Input: `[-2, 1, -3, 4, -1, 2, 1, -5, 4]`
Output: `[4, -1, 2, 1]` with a sum of `6`
My approach to solving this problem involves using Kadane's algorithm, which scans the array and at each position, it decides whether to continue the current subarray or start a new one. Here's my solution in Python:
```python
def max_subarray(arr):
    max_sum = float('-inf')
    current_sum = 0
    start = 0
    end = 0
    temp_start = 0
    
    for i in range(len(arr)):
        current_sum += arr[i]
        
        if current_sum > max_sum:
            max_sum = current_sum
            start = temp_start
            end = i
            
        if current_sum < 0:
            current_sum = 0
            temp_start = i + 1
            
    return arr[start:end+1], max_sum

# Example usage
arr = [-2, 1, -3, 4, -1, 2, 1, -5, 4]
subarray, max_sum = max_subarray(arr)
print("Maximum Subarray:", subarray)
print("Maximum Sum:", max_sum)
```
The time complexity of this solution is O(n), where n is the length of the input array, and the space complexity is O(1), as we only use a constant amount of space to store the temporary variables. This problem is a great example of how a simple and efficient algorithm can be used to solve a complex problem.
