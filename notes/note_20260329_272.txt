4. 💡 DAILY CODE PROBLEM
   - **Problem Title:** "Two Sum"
   - **Difficulty:** Easy
   - **Problem Statement:** Given an array of integers `nums` and an integer `target`, return indices of the two numbers such that they add up to `target`.
   - **Example Input:** `nums = [2, 7, 11, 15], target = 9`
   - **Example Output:** `[0, 1]` (because `nums[0] + nums[1] == 9`)
   - My approach was to use a hash table to store the numbers and their indices, and then iterate through the array to find the pair that adds up to the target.
   - Here's the full working solution in JavaScript:
     ```javascript
     function twoSum(nums, target) {
       const numToIndex = {};
       for (let i = 0; i < nums.length; i++) {
         const complement = target - nums[i];
         if (complement in numToIndex) {
           return [numToIndex[complement], i];
         }
         numToIndex[nums[i]] = i;
       }
       return null;
     }
     ```
   - Time complexity: O(n), where n is the length of the input array.
   - Space complexity: O(n), where n is the length of the input array.
