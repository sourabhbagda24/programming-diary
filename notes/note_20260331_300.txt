4. 💡 DAILY CODE PROBLEM 
   **Problem Title:** Maximum Subarray
   **Difficulty:** Easy
   **Problem Statement:** Given an integer array, find the contiguous subarray (containing at least one number) which has the largest sum and return its sum.
   **Example Input and Output:** Input: nums = [-2,1,-3,4,-1,2,1,-5,4], Output: 6
   **Approach:** I used a dynamic programming approach to solve this problem. 
   The idea is to iterate through the array and keep track of the maximum sum of the subarray ending at each position. 
   If the current sum is less than 0, I reset it to 0, as a negative sum is not desirable. 
   I keep track of the maximum sum seen so far and return it at the end.
   **Full Working Solution (Python):**
   ```
   def max_subarray(nums):
       if not nums:
           return 0
       max_sum = float('-inf')
       current_sum = 0
       for num in nums:
           current_sum = max(num, current_sum + num)
           max_sum = max(max_sum, current_sum)
       return max_sum
   ```
   **Time and Space Complexity:** The time complexity is O(n), where n is the length of the input array, and the space complexity is O(1), as I only use a constant amount of space to store the maximum sum and the current sum.
   I solved this problem using a simple iterative approach, which makes it efficient and easy to understand.
