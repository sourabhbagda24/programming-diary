### 💡 DAILY CODE PROBLEM
**Problem Title:** Reverse Linked List
**Difficulty:** Easy
**Problem Statement:** Given the head of a singly linked list, reverse the list and return the reversed list.
**Example Input:** 1 -> 2 -> 3 -> 4 -> 5
**Example Output:** 5 -> 4 -> 3 -> 2 -> 1
My approach to solving this problem involves using a simple iterative method. I initialize three pointers: previous, current, and next. I traverse the linked list, and for each node, I update the next pointer to point to the previous node. This effectively reverses the linked list.

Here is the full working solution in JavaScript:
```javascript
// Definition for singly-linked list.
function ListNode(val, next) {
  this.val = (val === undefined ? 0 : val);
  this.next = (next === undefined ? null : next);
}

/**
 * @param {ListNode} head
 * @return {ListNode}
 */
function reverseList(head) {
  let prev = null;
  let curr = head;
  while (curr !== null) {
    let nextTemp = curr.next;
    curr.next = prev;
    prev = curr;
    curr = nextTemp;
  }
  return prev;
}
```
Time complexity: O(n), where n is the number of nodes in the linked list.
Space complexity: O(1), since we only use a constant amount of space to store the pointers.
