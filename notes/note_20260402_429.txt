4. 💡 DAILY CODE PROBLEM 
   - Problem title: "First Duplicate" 
   - Difficulty: Medium 
   - Problem statement: Given an array of integers, find the first duplicate in the array. 
   - Example input: [2, 1, 3, 4, 2] 
   - Example output: 2 
   - Approach: I used a hash set to store the elements of the array as I iterate through it. 
   - If I encounter an element that is already in the hash set, I return that element as the first duplicate. 
   - Here is the full working solution in JavaScript:
     ```
     function firstDuplicate(arr) {
       const hashSet = new Set();
       for (let i = 0; i < arr.length; i++) {
         if (hashSet.has(arr[i])) {
           return arr[i];
         } else {
           hashSet.add(arr[i]);
         }
       }
       return -1;
     }
     ```
   - Time complexity: O(n), where n is the length of the array. 
   - Space complexity: O(n), where n is the length of the array. 
   - This solution works by iterating through the array and checking if each element is already in the hash set. 
   - If it is, we return that element as the first duplicate. 
   - If not, we add the element to the hash set and continue iterating.
