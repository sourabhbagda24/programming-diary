4. 💡 DAILY CODE PROBLEM 
   **Problem Title:** Reverse Linked List
   **Difficulty:** Easy
   **Problem Statement:** Given the head of a singly linked list, reverse the list and return the reversed list.
   **Example Input and Output:**
   Input: head = [1,2,3,4,5]
   Output: [5,4,3,2,1]
   My approach to this problem was to use a simple iterative method to reverse the linked list. I initialized three pointers: previous, current, and next. I set the previous pointer to null and the current pointer to the head of the list. Then, I iterated through the list, reversing the direction of the pointers as I went. Here is the full working solution in JavaScript:
   ```
   /**
    * Definition for singly-linked list.
    * function ListNode(val, next) {
    *     this.val = (val===undefined ? 0 : val)
    *     this.next = (next===undefined ? null : next)
    * }
    */
   /**
    * @param {ListNode} head
    * @return {ListNode}
    */
   var reverseList = function(head) {
       let prev = null;
       let curr = head;
       while (curr !== null) {
           let nextTemp = curr.next;
           curr.next = prev;
           prev = curr;
           curr = nextTemp;
       }
       return prev;
   };
   ```
   The time complexity of this solution is O(n), where n is the number of nodes in the linked list, and the space complexity is O(1) because we only use a constant amount of space to store the pointers.
   I chose this problem because it's a classic example of a linked list operation, and it's a great way to practice iterative solutions. The solution is also relatively simple, making it easy to understand and implement.
