4. 💡 DAILY CODE PROBLEM 
   Problem title: 'Valid Anagram'
   Difficulty: Easy
   Problem statement: Given two strings, determine if one is an anagram of the other.
   Example input: str1 = "listen", str2 = "silent"
   Example output: true
   Approach: To solve this problem, we can use a sorting algorithm to sort the characters in each string and then compare the sorted strings. If they are equal, then the original strings are anagrams of each other. 
   Another approach is to use a hash table to count the frequency of each character in the strings. If the frequency counts are equal, then the strings are anagrams.
   Here is the full working solution in JavaScript:
   ```javascript
   function isAnagram(str1, str2) {
       // Remove any white spaces and convert to lower case
       str1 = str1.replace(/\s/g, "").toLowerCase();
       str2 = str2.replace(/\s/g, "").toLowerCase();

       // If the lengths are not equal, return false
       if (str1.length !== str2.length) {
           return false;
       }

       // Create a hash table to count the frequency of each character
       let charCount = {};

       // Count the frequency of each character in str1
       for (let char of str1) {
           if (char in charCount) {
               charCount[char]++;
           } else {
               charCount[char] = 1;
           }
       }

       // Subtract the frequency of each character in str2
       for (let char of str2) {
           if (char in charCount) {
               charCount[char]--;
           } else {
               return false;
           }
       }

       // If all counts are zero, return true
       for (let count in charCount) {
           if (charCount[count] !== 0) {
               return false;
           }
       }

       return true;
   }

   console.log(isAnagram("listen", "silent"));  // Output: true
   ```
   Time complexity: O(n log n) due to the sorting, or O(n) using the hash table approach.
   Space complexity: O(n) for the hash table.
