4. 💡 DAILY CODE PROBLEM
   **Problem Title:** Validate Subsequence
   **Difficulty:** Medium
   **Problem Statement:** Given two arrays, determine if the second array is a subsequence of the first array. A subsequence is a sequence that can be derived from another sequence by deleting some elements without changing the order of the remaining elements.
   **Example Input and Output:**
   ```
   array = [5, 1, 22, 25, 6, -1, 8, 10]
   sequence = [1, 6, -1, 10]
   Output: True
   ```
   My approach to solving this problem was to use a two-pointer technique. I initialized two pointers, one at the beginning of the array and one at the beginning of the sequence. Then, I iterated through the array, and whenever I found an element that matched the current element in the sequence, I moved the sequence pointer forward. If I reached the end of the sequence, I returned True, indicating that the sequence is a subsequence of the array. If I reached the end of the array without finding all the elements in the sequence, I returned False.
   Here is the full working solution in JavaScript:
   ```javascript
   function isValidSubsequence(array, sequence) {
     let arrIdx = 0;
     let seqIdx = 0;
     while (arrIdx < array.length && seqIdx < sequence.length) {
       if (array[arrIdx] === sequence[seqIdx]) {
         seqIdx++;
       }
       arrIdx++;
     }
     return seqIdx === sequence.length;
   }
   ```
   The time complexity of this solution is O(n), where n is the length of the array, and the space complexity is O(1), since we only use a constant amount of space to store the pointers.
