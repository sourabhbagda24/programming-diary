4. 💡 DAILY CODE PROBLEM 
**Problem Title:** Validate Palindrome
**Difficulty:** Easy
**Problem Statement:** Given a string, determine if it is a palindrome, ignoring non-alphanumeric characters and case sensitivity.
**Example Input and Output:**
Input: "A man, a plan, a canal: Panama"
Output: True
Input: "Not a palindrome"
Output: False
My approach was to first remove all non-alphanumeric characters from the string, and then convert it to lowercase. I then compared the resulting string with its reverse, and if they were the same, I returned True, indicating that the string is a palindrome. Here is the full working solution in JavaScript:
```javascript
function isPalindrome(s) {
    let cleaned = s.replace(/[^a-zA-Z0-9]/g, '').toLowerCase();
    let reversed = cleaned.split('').reverse().join('');
    return cleaned === reversed;
}
```
Time complexity: O(n), where n is the length of the string.
Space complexity: O(n), where n is the length of the string.
This problem requires careful consideration of edge cases and a solid understanding of string manipulation in JavaScript.
