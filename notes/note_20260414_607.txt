4. 💡 DAILY CODE PROBLEM 
   - **Problem Title:** Reverse Linked List (Easy)
   - **Problem Statement:** Given the head of a singly linked list, reverse the list and return the reversed list.
   - **Example Input:** 1 -> 2 -> 3 -> 4 -> 5
   - **Example Output:** 5 -> 4 -> 3 -> 2 -> 1
   - **Approach:** I used a simple iterative approach to reverse the linked list, by keeping track of the previous node and updating the next pointer of each node.
   - **Full Working Solution (Python):**
     ```
     class ListNode:
         def __init__(self, x):
             self.val = x
             self.next = None

     def reverseList(head):
         prev = None
         while head:
             next_node = head.next
             head.next = prev
             prev = head
             head = next_node
         return prev
     ```
   - **Time Complexity:** O(n), where n is the number of nodes in the linked list.
   - **Space Complexity:** O(1), as we only use a constant amount of space to store the previous node.
   - I found this problem to be a great exercise in understanding linked lists and pointers, and it helped me practice my problem-solving skills.
